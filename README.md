# CppDataTypesOOP 

A C++ STL library to get object orientated classes for the rudementary datetypes +
String class for std::string

This is a low priority project.
--- 

## Idea
The idea is simply to have class types like in C# or Java.

## Planned classes
* Boolean
* Character
* Integer
* Float
* Double
* Valueless?
* WideCharacter
* String
